### Data Storage

### Security
The access of s3 bucket and dynamoDB need the roles with permission. For the api gateway we enabled CORS with the
header of our web application domain to avoid the api access from other web sites.

### CloudWatch
The CloudWatch has been using as the debug method of the backend lambda functions.
